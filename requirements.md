## Packages
framer-motion | Page transitions and complex animations for the pachinko events
lucide-react | High-quality icons for the UI
clsx | Utility for constructing className strings conditionally

## Notes
The application requires the following endpoints to be implemented in the backend:
- GET /api/sessions -> Returns the leaderboard of saved sessions
- POST /api/sessions -> Saves a new session score
